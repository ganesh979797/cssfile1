# HTML and CSS Practice Page

A simple, semantic HTML5 document designed to demonstrate various CSS selectors including **element tags**, **classes**, **IDs**, and **internal styling**.

---

## Page Overview
This project serves as a foundational layout featuring:
* **Headings:** A hierarchy from `<h1>` to `<h3>`.
* **Typography:** Multiple paragraphs of placeholder text.
* **Lists:** Examples of both unordered (`<ul>`) and ordered (`<ol>`) lists.
* **Hyperlinks:** A styled anchor tag directed to Google.

---

## Styling Breakdown
The page utilizes a mix of styling methods:

### 1. Internal CSS
Targeting anchor (`<a>`) tags:
- **Color:** Chocolate
- **Weight:** Bold

### 2. External CSS
The file links to `style.css`. Ensure this file exists in your project folder.

### 3. Selectors Used
* **Classes (`.head`):** Used for heading consistency.
* **IDs (`#bglime`, `#para`):** Used for specific element targeting.

---

## File Structure
* `index.html` - Main structure.
* `style.css` - External styles.
